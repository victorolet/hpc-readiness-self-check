# HPC Readiness Self-Check

A small web app to help users determine whether their workload could benefit from HPC (High Performance Computing) resources.

## Features

- Single-page HTML/CSS/JavaScript app
- 7 binary questions (Yes/No)
- Automatically highlights HPC relevance when any answer is Yes
- Displays a call-to-action link to the Pawsey website

## Usage

1. Open `index.html` in a browser.
2. Answer the questions.
3. View the result message.

## Files

- `index.html` - the complete frontend UI and logic.
